## Run and Test
To run and test the assignment (Commandline)

Clone the repository
```
  git clone https://github.com/PPL-IIITA/ppl-assignment-gadabout-geek.git
```
Change the directory
```
  cd ppl-assignment-gadabout-geek/
```
Compile the files
```
  javac -d . *.java
```
